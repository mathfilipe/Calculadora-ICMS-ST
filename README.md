# Calculadora-ICMS-ST
Uma calculadora simples de ICMS-ST (Substituição Tributária) desenvolvida para facilitar o cálculo de operações tributárias de forma rápida e prática.

Funcionalidades:
Cálculo de ICMS-ST
Interface simples e intuitiva
Resultado automático da operação
Projeto leve e responsivo

 Tecnologias Utilizadas
HTML5
CSS3
JavaScript
